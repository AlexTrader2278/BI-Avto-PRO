## 🚀 Как Начать (Локально)

### 1. Клонируй репозиторий
```bash
git clone https://github.com/AlexTrader2278/BI-Avto-PRO.git
cd BI-Avto-PRO
```

### 2. Установи зависимости
```bash
npm install
```

### 3. Запусти проект
```bash
npm run dev
```

### 4. Открой в браузере
Отправься на [http://localhost:3000](http://localhost:3000)

## ❓ FAQ

**Q: Как добавить свой API ключ?**
A: Отредактируй `.env.local` файл с твоим ключом

**Q: Работает ли оффлайн?**
A: Нет, требуется интернет для API запросов

**Q: Можно ли изменить AI провайдера?**
A: Да, см. файл `lib/ai.ts` - там можно подключить Claude/другой сервис

**Q: Как восстановить стандартный API ключ?**
A: Ключ зашифрован в коде - контактируй администратора
